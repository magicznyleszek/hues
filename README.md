# Hues

A tool for working with colors. Build with Parcel, TypeScript and Vue.

Strongly inspired by awesome [Name that Color](http://chir.ag/projects/name-that-color) by Chirag Mehta &heart;.

## Building

1. `npm install`
2. `npm build-website`
3. open `docs/index.html`

## Development

1. `npm start` :)
